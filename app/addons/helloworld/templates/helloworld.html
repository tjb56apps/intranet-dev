Getting started

Struktur Folder

Buat folder baru di dalam folder "addons" nama folder sesuaikan dengan nama modul anda.

File dasar

Buat file dan folder baru di dalam folder yang sudah anda buat sebelumnya.

  • model
  • static
  • templates
  • routes.py
  • __init__.py
  • __manifest__.py

Model

Folder ini digunakan untuk menyimpan file model database

Cara penggunaan silahkan cek file helloworld.py pada folder model di modul helloworld ini.

Static

Folder ini digunakan untuk menyimpan file statik

{{'{{url_for("static", filename="addons/helloworld/static/your_file.js")}}'}}

Templates

Folder ini digunakan untuk menyimpan file template HTML

{{ '{% extends "base_template/base_merge_template_v3/base.html" %}{% block content %}your content{% endblock content %}' }}

Untuk menyesuaikan desain modul silahkan merujuk pada template base HTML.

Routes

File ini digunakan untuk membuat routing dan penempatan, pengintegrasian Business logic

from flask import render_template, redirect, url_for, session

from app.addons.helloworld import bp

from app.manifest import get_addon, get_child_path, get_title_module

....

Cara penggunaan route ini silahkan periksa file routes.py pada modul helloworld yang sudah kami sediakan.

Init

File ini digunakan untuk membuat Blueprint

from flask import Blueprint

bp = Blueprint("helloworld", __name__)

from app.addons.helloworld import routes

Manifest

File ini digunakan untuk membuat struktur menu berupa tipe data JSON

{ "id": "AAAA", "name": "Hello World", "version": "1.0", "path": "helloworld", "nav": [ { "id": "0", "type": "single", "icon": "fa-home", "name": "Home", "path": "", "leveling": ["parent", "member"] }, { "id": "1", "type": "dropdown", "icon": "fa-file-word-o", "name": "Settings", "leveling": ["parent", "member"], "child": [ { "name": "Sub menu 1", "icon": "fa-circle-o", "path": "sub-menu-1", "leveling": ["parent"] }, { "name": "Sub menu 2", "icon": "fa-circle-o", "path": "sub-menu-2", "leveling": ["parent", "member"] } ] }, ] }

Text warna merah artinya yang boleh anda rubah.

  • ID, Name, Version, Path sesuaikan dengan informasi modul anda.
  • Type [single] : Membuat menu tanpa dropdown
  • Type [dropdown] : Membuat menu dropdown
  • Leveling [parent] : untuk membuat kondisi level PARENT
  • Leveling [member] : untuk membuat kondisi level MEMBER

Koneksi Database

Membuat koneksi database baru di Extensions/database.py

engine = create_engine("sqlite:///project.db")

Session = sessionmaker(bind=engine)

db = Session()

Untuk penggunaan lebih lanjut periksa berkas yang sudah ada.

Akses Database Utama

Anda diperbolehkan mengakses database utama hanya untuk kepentingan tertentu. dan tidak diperbolehkan untuk merubah apapun di database utama.
